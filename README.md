# Color Pop (Tap-to-Fill Coloring App)

A mobile-friendly web coloring app where you **tap/click to fill** regions inside line art. Includes **undo/redo** and **export to PNG**. Built with **React + Vite** and deploys to **GitHub Pages** via **GitHub Actions**.

## Features
- Tap/click to flood-fill a region
- Outline-aware fill (wonâ€™t cross dark lines)
- Color palette
- Undo / Redo
- Export finished art as a PNG
- Works on mobile (touch support)
